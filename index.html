Programming Overview

In our programming class, we are learning about what HTML is and what CSS is and the different parts that make them (tags, elements, links, etc.).

HTML

HTML stands for Hypertext Markup Language. It is a basic way of creating a website on the world
wide web using tags and hyperlinks. The following is a list of tags that one might use to create a website:

  • <head></head>: this is the head of the HTML. It gives basic information
    for the document such as title, meta info., and style!
  • <body></body>: this is the body element. It is responsible for all of the
    information that is visually displayed on a webpage.
  • <h1>-<h6>: this is the header tag. <h1> is for the biggest header and
    <h6> is for the smallest.
  • <p></p>: these are opening and closing paragraph tags. They are used for
    chunks of text on a webpage.
  • <table></table>: this is a table tag. It can be used to organize information.
  • <ul></ul> and <ol></ol>: <ul> stands for "unordered list".
    It will create a list with bullet points. On the other hand, <ol>,
    which stands for "ordered list", will create a numbered list.
  • <img src= “” width= “” alt= “”/>: this is
    how one would link an image to a website.
  • <a href= “”> link </a>: this is how one would link one website
    to another website.

CSS

CSS stands for Cascading Style Sheet. There are three ways to incorporate CSS into an HTML file:

  • an external style sheet: this is when you link a CSS file into an HTML file, allowing one to
    create the CSS style sheet for multiple pages.
  • an internal style sheet: this is when the CSS is put into the head of an HTML file through
    <style></style>, creating styling specific to the webpage.
  • an inline style attribute: this would go into the opening tag of an element and it
    would have styling specific to the element it is in.
